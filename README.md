# Dungeon-Crawling-Game
About the Project
I created a simple dungeon crawling game (inspired by Slay the spire), I built as a learning project. The goal is to explore a dungeon, fight enemies, and collect treasures. It’s not a professional game, but a fun way for me to practice programming and game development basics. I use Dijkstra's algorithm and A* algorithm as the core of the game

Features
- Move around the dungeon map
- Battle enemies with basic combat system
- Customizing stage
- special effect node

Why I Made This
I’m a first-semester university student, and this project helps me learn coding step by step. It’s a mix of fun and practice, so don’t expect a perfect game,  but I hope you enjoy trying it out!
